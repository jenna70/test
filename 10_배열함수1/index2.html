배열함수1

자바스크립트에서 배열은 탐색, 수정, 삭제작업을 수행하기 위한 다양한 함수(메서드)를 제공한다.

  1. 배열명.shift() : 배열에서 첫번째 원소(맨 앞에 있는 원소)를 제거하기
  2. 배열명.unshift() : 배열에서 첫번째 원소에 추가
  3. 배열명.pop() : 배열에서 마지막번째 요소를 제거
  4. 배열명.push() : 배열에서 마지막번째 요소를 추가
  5. 배열명.splice(요소를 넣을 공간의 뒷 index값,0(삽입할때는 0, 삭제할때는 1),삽입할 요소) : 배열의 중간에서 요소를 추가.제거
  6. 배열명.join() : 배열의 값 사이에 원하는 문자를 삽입
  7. 배열명.concat() : 2개의 배열을 하나로 합치기
  8. 배열명.reverse() : 배열을 역순으로 재배치
  9. 배열명.sort() : 배열의 값을 순서대로 정렬
  10. 배열명.includes() : 배열의 해당 값의 존재여부를 true/false로 확인
  11. 배열명.indexOf() : 배열의 색인값(index=key값)을 찾아줌
  12. 배열명.slice(시작index, 끝index) : 배열에서 시작index부터부터 끝index(끝index는 미포함)까지의 배열을 복사해서 리턴
  13. 배열명.splice(시작점, 갯수) : 배열에서 시작점, 갯수만큼 삭제
